Angelについて

聖三一幼稚園の先生と一緒に、遊具で遊んだり、親子でリズム体操・物を作る制作遊び・絵本の読み聞かせなどを行っています。
又、劇団や歌のお兄さんに来ていただく特別な会もあります。遊びながら、子育て相談にも随時お受けいたします。
聖三一幼稚園の設備や、雰囲気を知っていただく為の機会にもなりますので、是非おこしください。毎月2回～3回のイベントを用意しております。

Angel Babyについて

Angel Babyでは0歳児～１歳児のお子様をお持ちの方、妊婦の方を対象にゆったりとしたお家の様な空間でリラックスをしてゆっくり過ごします。赤ちゃんとの遊び方・お話の仕方・お歌の聞かせ方・絵本の読み方・ベビーマッサージ・ベビーダンスなど、ご一緒に子育てを楽しみましょう。子育てについて、専門家がレクチャーさせて頂きます！
ご一緒するのは、亀ちゃん先生（大谷大学 乳児保育概論講師・亀ちゃん文庫絵本の先生）と、園長みゆき先生（NPO法人 国際隣保保育研究所認定foundation skills program insutructor ・大阪府私立幼稚園連盟認定子育て支援相談員）です。
